Get Set Up
Two API keys to connect — takes about 10 minutes. You only do this once. Keys are saved in your browser.
✅
Step 1 of 2
Google Places API Key
This lets Boris.AI search for real businesses by sector and location across the UK.
  1. Go to console.cloud.google.com and sign in with your Google account
  2. Click Create Project → name it "Boris AI" → click Create
  3. In the left menu go to APIs & Services → Library
  4. Search for "Places API" → click it → click Enable
  5. Go to APIs & Services → Credentials
  6. Click + Create Credentials → API Key → copy the key shown
  7. Paste it below. Google gives you $200 free credit/month — enough for hundreds of searches
✅
Step 2 of 2
HubSpot Private App Token
This lets you push companies directly into HubSpot as contacts with one click — only when you choose to.
  1. In HubSpot, click your account name (top right) → Account Settings
  2. Go to Integrations → Private Apps in the left menu
  3. Click Create a private app
  4. Name it "Boris AI" → go to the Scopes tab
  5. Enable: crm.objects.companies.write and crm.objects.contacts.write
  6. Click Create app → copy the token shown (starts with pat-)
  7. Paste it below
